# Carpeta outputs/

Aqui se guardan automaticamente los archivos generados por los scripts
(no se necesita crearlos a mano, los scripts crean esta carpeta si no existe):

- modelo_dualencoder_dino_resnet.pth  -> pesos del modelo entrenado (train.py)
- embeddings.npy                      -> embeddings de las imagenes (embeddings.py)
- embeddings_nombres.txt              -> nombres correspondientes a cada embedding
- metricas_dualencoder.png            -> graficas de evaluacion (test.py)
